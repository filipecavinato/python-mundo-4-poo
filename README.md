# 🐍 Curso de Python - Curso em Vídeo

Repositório dedicado aos exercícios e desafios propostos pelo professor Gustavo Guanabara no Curso em Vídeo. O objetivo aqui é praticar lógica de programação, sintaxe e estruturas de dados.

## 📂 Sobre este Módulo

Este repositório contém as resoluções dos desafios do **Mundo 4**, dedicado ao paradigma de **Programação Orientada a Objetos (POO)**. Este módulo representa uma mudança fundamental na forma de estruturar projetos, focando em modelar problemas do mundo real:
* **Classes e Objetos:** A base da abstração em Python.
* **Atributos e Métodos:** Como definimos estados e comportamentos.
* **Construtores (`__init__`):** Inicialização inteligente de objetos.
* **Pilares da POO:** Implementação de Encapsulamento, Herança e Polimorfismo.
* **Arquitetura:** Organização de sistemas complexos e escaláveis.

## 🚀 Como executar

Como são scripts individuais, cada arquivo .py pode ser executado diretamente pelo terminal:

```
python pasta_do_exercicio/nome_do_arquivo.py
```

## 🛠️ Tecnologias

- Linguagem: Python 3.x
- Ambiente: PyCharm
- Bibliotecas: `random`, `time`, `datetime`, `rich`, `os`
- Versionamento: Git & GitHub
